Specification

POST friendships/create

Create a friend connection between two email addresses.

Resource URL

https://sp-friends-api.cfapps.io/friendships/create

Fields

Name

Description

Required

friends

A list of two email address.

Y

Request Example

{

  friends:

    [

      'andy@example.com',

      'john@example.com'

    ]

}

Response Example

{
  "success": true
}

Failure Example

{
  "success": false
   "errorMessage": “No friends list provided.”
}

POST friendships/searches

Retrieve the friends list for an email address.

Resource URL

https://sp-friends-api.cfapps.io/friendships/searches

Fields

Name

Description

Required

email

Email address of the requestor

Y

 

Request Example

{
  email: 'andy@example.com'
}

 

Response Example

{
  "success": true,
  "friends" :
    [
      'john@example.com'
    ],
  "count" : 1   
}

 

Failure Example

{
  "success": false,
   "errorMessage": “No email provided.”
}

POST subscription/searches

Retrieve all email addresses that can receive updates from an email address.

Resource URL

https://sp-friends-api.cfapps.io/subscription/searches

Fields

Name

Description

Required

sender

Email address of the sender

Y

text

Text to be posted

Y

 

Request Example

{
  "sender":  "john@example.com",
  "text": "Hello World! kate@example.com"
}

 

Response Example

{
  "success": true
  "recipients":
    [
      "lisa@example.com",
      "kate@example.com"
    ]
}

 

Failure Example

{
  "success": false
  "errorMessage": “No sender email provided.”
}
